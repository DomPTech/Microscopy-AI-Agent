# Microscopy-AI-Agent-Demo
This research is designed to facilitate autonomous control of Scanning Transmission Electron Microscopes through utilization of a local LLM. The specific machine that this software is designed for is the ThermoFisher Spectra 300 TEM. 

## Project Diagram

![Project Diagram](/Images/Diagram_for_STEM_Model.jpg)

## Tasks
* <u>Define array of functions</u>
* Determine Local model type with sufficient tool calling capabilities
* Build the whole thing
* Train model using digital twin servers then transition to actual servers
* Create a GUI for fluid user interaction
